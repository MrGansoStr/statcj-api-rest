# About this project
This project is used for StatCJ with ReactJS to manage comments and users.

# Install dependencies
> npm install  
> npm i

## Run code
In Developer Mode is used with Morgan and you can supervise all request.

### Developer Mode:
> npm run dev
### Normal Mode:
> npm start

## Build
> npm build

## Technologies | Libraries
- **NodeJS - Express**
- **Mongoose**
- **Bcryptjs**
- **Cors**
- **Dotenv**
- **JsonWebToken**
- **Morgan**
